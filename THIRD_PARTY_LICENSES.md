## Piano MP3 Samples

- Package/repository: `fuhton/piano-mp3`
- Source: https://github.com/fuhton/piano-mp3
- Upstream source referenced by repository: https://github.com/gleitz/midi-js-soundfonts
- License: MIT

Piano360 bundles local MP3 samples for the `A3-C5` practice keyboard range. The app displays black keys as sharps, but stores the black-key MP3 files using the source repository's URL-safe flat-note filenames, for example `Bb3.mp3` for `A#3`.
